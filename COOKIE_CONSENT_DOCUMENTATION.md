#### Функционал:
- Модальное окно появляется при первом посещении сайта

- Содержит ссылки на политику конфиденциальности и условия использования
- Упоминание о Яндекс.Метрике

## Как это работает

1. **При первом посещении сайта:**
   - Пользователь видит модальное окно с предложением принять политику cookies
   - Окно блокирует взаимодействие с сайтом до принятия решения

2. **После принятия:**
   - Создается cookie `cookie_consent=true` на 365 дней
   - Модальное окно скрывается
   - Можно инициализировать дополнительные системы аналитики

3. **После отклонения:**
   - Создается cookie `cookie_consent=false` на 30 дней
   - Модальное окно скрывается
   - Через 30 дней окно появится снова

4. **Footer на всех страницах:**
   - Предоставляет доступ к важным страницам
   - Отображает контактную информацию
   - Ссылки на социальные сети

## Следующие шаги

1. Скомпилируйте frontend: `npm run build`
2. Очистите кеш: `php artisan cache:clear`
3. Проверьте работу на сайте
4. При необходимости настройте инициализацию аналитики в методе `initializeAnalytics()` класса `CookieConsent`
